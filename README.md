## 실행방법

> https://github.com/Myunggyun/ioCrops_react 에서 코드를 clone한다.
> command line 에서 ~/iocrops-react$ npm install 를 실행한다.
> command line 에서 ~/iocrops-react$ npm start 를 실행한다.

## 구현내용

> 시계열 Line 차트 구현
> Brush and Zoom 기능
> 각 데이터들의 이름, 평균값, 차트에 데이터를 표시할지 말지를 결정해주는 버튼을 포함한 차트 테이블 구현
>
> > legend 클릭시 데이터를 표시하는 기능이 있어서 모든 데이터 표시 삭제 기능 구현

> 반응형
>
> > 화면 크기 줄였을 때, legend 위치 변경, x축 날짜 줄임

## stack

> React Hoook, typescript, styled-componenets, highchart
